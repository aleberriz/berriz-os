# berriz-os

Personal brand operating system for [Alejandro Berrizbeitia](https://github.com/aleberriz).

This repository is the public layer of a system that indexes projects, holds the narrative
canon, tracks open-source collaboration targets, and hosts the source for
[aleberriz.com](https://aleberriz.com).

---

## What's here

| Folder | Contents |
|--------|----------|
| [`brand/`](brand/) | Positioning, bios, voice guide, FOSS posture — the source of truth for all public copy |
| [`roadmap/`](roadmap/) | The multi-phase personal brand plan with rationale for every decision |
| [`projects/`](projects/) | Full repo index (`projects.yaml` is the machine-readable source; `index.md` is the view) |
| [`collaborations/`](collaborations/) | OSS contribution targets, prioritized with goals and status |
| [`website/`](website/) | Source for [aleberriz.com](https://aleberriz.com) (Astro + MDX + Tailwind) |
| [`workspaces/`](workspaces/) | `.code-workspace` files for multi-repo Cursor sessions |
